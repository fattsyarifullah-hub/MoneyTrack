# Project name

**MONEYTRACK**

## Description
tracking your money with simple features, suitable for keeping simple financial records

## Getting Started
1. Clone the repository
```bash
git clone https://github.com/fattsyarifullah-hub/MoneyTrack.git
```

2. Run :
```bash
flutter pub get
```

3. Start the app :
```bash
flutter run 
```
